## Overview

Create animated React components for video slides including title cards, argument displays, charts, and transitions. Use Remotion's interpolate API for smooth animations.

---

## Objectives

1. Animated slide components
2. Smooth transitions (fade, slide, zoom)
3. Text animations (typewriter, reveal)
4. Chart/graph animations
5. Brand-consistent styling

---

## Key Components

- TitleSlide: Fade-in title with subtitle
- ArgumentSlide: Slide-in argument cards
- ModeratorSlide: Split-screen synthesis
- TransitionSlide: Section dividers
- ChartSlide: Animated bar/pie charts
